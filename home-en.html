AVRConfig is simple multiplatform graphical interface for the program avrdude written in JavaFX. It was developed to make working with AVR microcontrollers easier (primarily for people who don't like console). It was also developed for those who did their projects with Arduino or with other simple SW and HW and want to try programming in pure C with only classical HW. The application uses the program avrdude by Brian S. Dean.


Note: we recommend you to download and read our documentation of the program to introduce you all the posibilities or the limmitations of the program. If you found some mistakes in the program, documentation, or on this site, we would be pleasured if you could inform us via our mail lamyvkabatu@gmail.com or our Git


Functions:

  • -Upload to/read from flash memory or EEPROM.
  • -Write/read fuse bits.
  • -Write/read lock bits.
  • -Check the microcontroller connection.
  • -Erase the whole microcontroller.


There are 2 versions for download:


the first one - contains AVRConfig only (so you have to download avrdude program and avrdude configuration file),


the second one - contins programs AVRConfig, avrdude.exe and avrdude configuration file.
